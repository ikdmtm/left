## 2. 説明文（4000文字以内）

```
「Left」は、人生の残り時間を可視化し、時間を大切にするきっかけを作るアプリです。

■ 主な機能

【残り時間の表示】
統計上の平均寿命または設定した寿命までの残り時間を、複数の単位で確認できます。
• 秒、日、週、月、年
• YYYYMMDDHHMMSS形式
• 到達予定日時

【人生の進捗バー】
人生全体の進捗を視覚的に表示。あなたの人生の「今」を一目で確認できます。

【今日の活動時間】
設定した活動時間（起床〜就寝）の残り時間を表示。今日という日がどれだけ残っているかを意識できます。

【1行メモ】
今日または今週の目標や気づきを1行で記録。シンプルだから続けられます。

■ このアプリの特徴

• オフライン完結：すべてのデータは端末内に保存され、外部サーバーに送信されません
• プライバシー重視：個人情報の収集は一切ありません
• シンプルなUI：必要な情報だけを、見やすく表示
• 通知なし：あなたのペースで確認できます

■ こんな方におすすめ

• 時間を大切にしたい
• 目標達成に向けて意識を高めたい
• 日々の優先順位を見直したい
• シンプルなライフログをつけたい

■ 注意事項

このアプリは統計上の目安を表示するものであり、寿命を予測するものではありません。
人生の長さは誰にもわかりません。このアプリは、今という時間の大切さを再認識するためのツールです。

■ プライバシーとセキュリティ

• データ収集なし：個人情報は一切収集しません
• ローカルストレージのみ：すべてのデータは端末内に保存
• 第三者提供なし：データを第三者と共有することはありません
• 追跡なし：ユーザーの行動を追跡しません

詳細なプライバシーポリシーは、アプリ内またはサポートサイトからご確認いただけます。

あなたの人生の「残り」を意識することで、「今」を大切に生きるきっかけになれば幸いです。
```

## 6. スクリーンショット

### 必要なサイズ

#### ✅ 重要: 1つのデバイスで撮影すればOK！

**iPhone 12をお持ちの場合**:
- iPhone 12で撮影したスクリーンショット（1170×2532px）をアップロード
- Appleが自動的に他のサイズ用に調整してくれます
- **複数の実機は不要です！**

#### 対応サイズ（参考）
- **6.9インチ（iPhone 16 Pro Max）**: 1320×2868px
- **6.7インチ（iPhone 15 Plus）**: 1290×2796px
- **6.5インチ（iPhone 11 Pro Max）**: 1242×2688px
- **6.1インチ（iPhone 12）**: 1170×2532px ← **あなたのデバイス**
- **5.5インチ（iPhone 8 Plus）**: 1242×2208px

**最低必要枚数**: 3〜5枚（同じデバイスで撮影）

#### iPad（iPadに対応している場合）
- **12.9インチ（iPad Pro）**: 2048×2732px または 2732×2048px
- **11インチ（iPad Pro）**: 1668×2388px または 2388×1668px

### スクリーンショットの撮り方

**方法1: Expo/React Native Devツールでシミュレータのスクショを撮る**

1. iOSシミュレータでアプリを起動
2. 必要な画面を表示
3. `Cmd + S` でスクリーンショットを撮る
4. デスクトップに保存される

**方法2: デバイスで撮影（推奨）**

1. TestFlightからインストールしたアプリで撮影
2. 各画面のスクリーンショットを撮る
3. 写真アプリからMacに転送

**方法3: App Store Connectのメディアマネージャー**

1. TestFlightビルドから自動生成されたスクリーンショットを使用
2. ただし、実際の使用画面ではないので推奨しない

### 必要なスクリーンショット（推奨）

1. **ホーム画面** - 残り時間表示、人生バー、活動時間
2. **設定画面** - 生年月日、寿命、活動時間の設定
3. **1行メモ機能** - メモ入力画面
4. **単位切り替え** - 秒/日/週/月/年の切り替え
5. **進捗バー** - 人生の進捗バーのアップ

**ヒント**:
- 明るい画面で撮影
- データが入力されている状態で撮影（空の画面は避ける）
- テキストが読めるサイズで撮影
- ステータスバーを含めて撮影（時計、バッテリーなど）

### スクリーンショットの編集（オプション）

- デザインツール（Figma、Canvaなど）でフレームを追加
- キャッチコピーを追加
- 機能説明を追加

**注意**: 過度な装飾は審査で却下される可能性があるため、シンプルに

## 審査プロセス

### タイムライン
1. **提出**: 即座
2. **審査待ち**: 通常24〜48時間
3. **審査中**: 通常24時間以内
4. **承認/却下**: 審査完了後に通知

### ステータス
- **提出準備中** → **審査待ち** → **審査中** → **承認済み** → **販売準備完了** → **App Storeで販売中**

### 却下された場合
1. 却下理由を確認
2. 必要な修正を実施
3. 再提出

---

## よくある却下理由と対策

### 1. スクリーンショットの問題
- **問題**: 実際のアプリと異なる
- **対策**: 実際のアプリのスクリーンショットを使用

### 2. プライバシーポリシーの問題
- **問題**: プライバシーポリシーにアクセスできない
- **対策**: URLが正しいか確認、HTTPS必須

### 3. 機能が不明瞭
- **問題**: アプリの機能がわからない
- **対策**: レビューノートに詳細な説明を追加

### 4. クラッシュやバグ
- **問題**: アプリがクラッシュする
- **対策**: TestFlightで十分にテスト

### 5. メタデータの問題
- **問題**: 説明文とキーワードが一致しない
- **対策**: 説明文とキーワードの整合性を確認
