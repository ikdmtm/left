# マイルストーン

## M1（最速で価値）
- Onboarding（初回）：生年月日、寿命年数、活動時間を設定して保存
- Home：
  - 残り時間（秒/日/週/月/年/YY...形式切替）
  - 人生バー
  - 活動時間の残り（今日）
  - 1行メモ（Today/Week切替 + 保存）

受入条件：
- アプリ起動→Homeで即見れる
- 設定を変更→Homeへ戻ると即反映
- オフラインで動作

## M2（品質）
- フォーマット改善（桁区切り、0埋め、到達予定日時の見やすさ）
- 設定画面UI改善（入力バリデーション）
- コア計算のユニットテスト追加

## M3（可視化強化）
- Weekグリッド（人生を週でドット表示）
- スムーズな切替アニメ（軽量）

## M4（配布・仕上げ）
- アイコン/スプラッシュ
- 設定エクスポート（JSON）※任意
- ストア向け文言整備
