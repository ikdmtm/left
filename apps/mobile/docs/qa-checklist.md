# 手動QAチェックリスト

## 初回導線
- [ ] 初回起動で設定画面（または簡易オンボーディング）が出る
- [ ] 生年月日を入れないと進めない（またはエラーが出る）
- [ ] 設定を保存するとHomeに遷移

## Home（残り時間）
- [ ] 秒表示が1秒ごとに更新される
- [ ] 秒以外は頻繁に更新されない（1分更新）
- [ ] 残りが0を下回らない（寿命到達後は0固定）
- [ ] 単位切替が正しく反映される
- [ ] YYYYMMDDHHMMSS 表示が0埋めされる

## 人生バー
- [ ] progress が 0〜100% の範囲
- [ ] 未来（誕生前など異常値）でも破綻しない

## 活動時間の残り
- [ ] activeStart〜activeEnd の範囲内で正しく減る
- [ ] 活動時間外は0になる
- [ ] start < end 前提（逆転入力は設定側で防止）

## 1行メモ
- [ ] Today/Weekの切替で保存先が変わる
- [ ] 再起動しても保持される
- [ ] 改行が入らない（貼り付けも含む）

## Settings
- [ ] 設定変更→Home反映
- [ ] 異常入力（寿命=0, 過大）を弾く or 最低値に丸める
