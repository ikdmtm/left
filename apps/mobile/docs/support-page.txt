■ GitHub Pagesへのアップロード手順
==================================================

1. GitHubで新しいリポジトリを作成
   - リポジトリ名: left-support
   - Public に設定

2. index.html を作成
   - 上記のHTMLコードをコピー
   - index.html として保存

3. GitHubにプッシュ
   git init
   git add index.html
   git commit -m "Add support page"
   git branch -M main
   git remote add origin https://github.com/ikdmtm/left-support.git
   git push -u origin main

4. GitHub Pagesを有効化
   - リポジトリの Settings → Pages
   - Source: Deploy from a branch
   - Branch: main / (root)
   - Save

5. 完成！
   サポートURL: https://ikdmtm.github.io/left-support/
